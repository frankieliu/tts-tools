METHODOLOGY
--------------------------------------------------------------------------------

Analysis Process:
1. JSON parsing of complete save file
2. Recursive object extraction (including nested objects)
3. Metadata extraction (GUID, type, position, properties)
4. Categorization by multiple dimensions:
   - Object type (TTS native types)
   - Functional purpose (game role)
   - Spatial zone (table position)
   - Color association (player colors)
5. Statistical aggregation
6. Report generation

Categorization Logic:
- Type-based: Uses TTS object type names
- Purpose-based: Infers function from type, nickname, and position
- Zone-based: Calculates from X/Z coordinates
- Color-based: Extracts from object nicknames

Tools Used:
- Python 3 with JSON library
- Custom analysis scripts
- Position-based spatial analysis
- Hierarchical object traversal
